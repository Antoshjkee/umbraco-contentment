# Umbraco Contentment

> contentment /kənˈtɛntm(ə)nt/ - a state of happiness and satisfaction


## What's on offer?

### Data Editors

> Data Editors appear to be the new v8 term for combining what was previously v6/v7 Property Editors and Parameter Editors

- Checkbox
- Dropdown
- Entity Picker
- Macro Picker
- Notes
- Render Macro
